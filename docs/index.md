A Python Library using BoardGameGeek's BGG XML API2 for running extract, store and transform jobs with configurablue storage options (on-device/cloud) and schemas. Full documentation is provided on BGG's official [website](https://boardgamegeek.com/wiki/page/BGG_XML_API2#toc17). Initially built based on the user guide and Terms of Use as on 23-Nov-2025.
